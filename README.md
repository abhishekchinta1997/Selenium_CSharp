# 🚀 Selenium C# Automation Framework

A Selenium WebDriver Automation Framework developed using C#, NUnit, and .NET technologies.

This project demonstrates web UI automation concepts, framework design principles, reusable components, and best practices commonly used in automation testing projects.

Perfect for:

- QA Automation Engineers
- Selenium Learners
- .NET Developers exploring Automation Testing
- Interview Preparation

---

## 📌 Project Overview

This repository contains automated test scripts and framework components built using:

- Selenium WebDriver
- C#
- NUnit
- .NET Framework
- WebDriverManager
- Page Object Model (POM)
- Data-Driven Testing Concepts

The framework is designed to be scalable, maintainable, and reusable.

## ▶️ Getting Started
Prerequisites
  1.	Visual Studio
  2.	.NET SDK
  3.	Chrome Browser
  4.	NuGet Packages
  5.	Clone Repository
  6.	git clone https://github.com/abhishekchinta1997/Selenium_CSharp.git
  7.	Restore Dependencies
  8.	dotnet restore
  9.	Build Project
  10.	dotnet build
  11.	Execute Tests
  12.	dotnet test
